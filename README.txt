== Description ==

Displays delivery time for a product on Woocommerce.

== Installation ==

1. Upload `delivery-time-for-woo` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure general settings of the plugin in Wordpress
